# Инструкция по запуску проекта CareSync

## Быстрый старт

1. **Установите зависимости:**
```bash
npm install
```

2. **Запустите проект в режиме разработки:**
```bash
npm run dev
```

3. **Откройте в браузере:**
```
http://localhost:3000
```

## Структура проекта

```
medical/
├── app/                    # Next.js App Router
```

## Особенности реализации

### Glassmorphism (iOS 26 стиль)
- Используются классы `.glass` и `.glass-dark` для эффекта стекла
- Backdrop-filter с blur эффектом
- Полупрозрачные фоны с границами

### Анимации
- Framer Motion для плавных переходов
- Анимации появления элементов
- Анимации нажатий (whileTap)

### Хранение данных
- Все данные сохраняются в localStorage
- Пользователь, записи, метрики здоровья
- Сохранение текущего экрана

### Адаптация для Telegram Mini App
- Safe-area для безопасных зон
- Отключение масштабирования
- Оптимизация для мобильных устройств

## Функционал экранов

### Splash Screen
- Анимированный логотип CareSync
- Автоматический переход на экран регистрации

### Sign Up / Log In
- Формы регистрации и входа
- Социальные кнопки (Facebook, Google)
- Валидация паролей

### Home
- Поиск по лекарствам и больницам
- Приветствие пользователя
- Быстрые действия (My doctor, Reports, Find Clinic, Appointment)
- Информация о следующем приеме

### Pharmacy
- Поиск лекарств и лабораторных тестов
- Промо-карточки (скидки, лабораторные тесты)
- Сканирование рецепта
- Категории товаров

### Appointment
- Каталог врачей с рейтингами
- Фильтрация по специализации
- Поиск врачей
- Календарь записи с выбором даты и времени

### Reports
- Доступ к лабораторным тестам
- Просмотр отчетов

### Profile
- Информация о пользователе
- Health Overview (Heart Rate, Blood Sugar, Blood Pressure)
- Ближайшая запись
- Напоминания о таблетках
- Меню настроек
- Выход из аккаунта

## Деплой на Vercel

1. Подключите репозиторий к GitHub
2. Импортируйте проект в Vercel
3. Настройте автоматический деплой
4. Приложение будет доступно по адресу: `https://your-project.vercel.app`

## Тестирование

Для тестирования функционала:
1. Зарегистрируйтесь или войдите
2. Создайте запись к врачу
3. Проверьте сохранение данных в localStorage
4. Проверьте навигацию между экранами
5. Проверьте анимации и переходы

## Примечания

- Все данные хранятся локально в браузере
- Для продакшена рекомендуется подключить backend API
- Telegram Mini App требует настройки через BotFather
- Для работы в Telegram нужно добавить скрипт инициализации Telegram Web App
